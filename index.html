SALIKSIK: UPHSL Research Repository

Thesis

A web-based Institutional Repository for disseminating, archiving and preserving digital copies of academic papers of University of Perpetual Help System Laguna - Biñan Campus.

Tech Stack:

  • HTML
  • CSS
  • JavaScript
  • BootStrap
  • JQuery
  • PHP
  • MySQL

User Features:

  • Register / Sign-in (University Account only)
  • Add/Edit/Delete Bookmarked Theses
  • Upload .pdf files of thesis for Admin approval
  • Search Results using various filters such as by Title, Description, Author, or Year
  • Update user profile

Admin Features:

  • Add/Edit/Delete submissions
  • Hide submissions from regular users
